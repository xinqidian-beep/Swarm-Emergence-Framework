# Swarm-Emergence-Framework

**一个以个体三元自锁为核心的Swarm Emergence世界模型框架**

让游戏NPC从脚本驱动进化成真实社会：每个独立Agent拥有长短期记忆，通过**三元自锁通用模板**实现自洽自锁、柔性稳固与双向通信。集成注意力机制节省算力，每5 Tick强化清除冗余。

## 核心特性
- **最底层通用模板**：三元自锁（Drive→Constraint→Integrate）已完整整合长短期记忆自洽自锁柔性稳固架构双向通信
- **真实涌现**：极化1.8~2.55 + 社区事件链自然触发
- **高效算力**：注意力机制 + 定期清除冗余
- **零依赖单HTML Demo**：浏览器直接打开 `swarm_emergence_demo.html` 即可体验个体三元自锁报告（带Physique恢复、delta变化、注意力统计）

## 快速开始
1. 打开 `swarm_emergence_demo.html`
2. 调整参数面板 → 点击启动仿真 → 观察每个Agent的三元自锁报告与事件链

## 项目全景
我们从MiroFish涌现机制出发，结合Harness柔性自锁，构建出以个体三元自锁为原子引擎的世界模型。历史所有设计思路已完全具身。
